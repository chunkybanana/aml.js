Aml.js

Aml.js is a framework that allows editing of AML. AML is a language similar to css, but uses attributes instead of styling.

Tutorial

Aml syntax is identical to that of css, except it uses attributes instead of styling. It must be inside a <div id="aml"> element, otherwise the Javascript parsing will not work.

<div id="aml">
input{
type: range;
min: 50;
max: 100;
}
</div>

All css syntax will work, including pseudo-classes, multiple selectors and *.

Use

Either download from github or host from https://amljs.surge.sh/aml.js and https://amljs.surge.sh/aml.css.
